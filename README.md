# TIS Task

## Requirements

- Node v16.15.0
- NPM v8.5.5
- PSQL


## How to run

- Download all packages: npm i
- create .env and add all environment values
- Run the app: npm run dev

# environment

````
PORT=3000
DB_HOST=localhost
DB_USERNAME=postgres
DB_PASSWORD=123456
DB_NAME=tis
DB_PORT=5432
NODE_ENV=development
HASH_SALT=TIS-Task-Hash-Salt
SECRET_KEY=TIS-Task-SECERET-KEY
````
